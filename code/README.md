# 文本分析可视化看板 (Text Analysis Dashboard)

## 项目简介

本项目是一个基于 `Flask` + `ECharts` + `Tailwind CSS` 构建的文本挖掘与可视化看板。系统通过自然语言处理（NLP）技术对输入文本进行解析，提供关键词提取、共现分析以及无监督聚类功能，并将分析结果以交互式网页图表的形式呈现。

---

## 主要功能

1. **关键词提取与可视化**
   - 基于 TF-IDF 算法提取文本核心词汇。
   - 提供词云图 (WordCloud)、词频分布（柱状图）与权重占比（饼图）三种基础可视化视图。

2. **共现网络分析**
   - 以句子为滑动窗口，计算核心词汇的共现频次。
   - 通过力导向图 (Force Directed Graph) 展示词语间的关联网络，辅助发现潜在的话题组合。

3. **语义聚类分析**
   - 使用 `TfidfVectorizer` 构建词汇特征向量，并结合 `K-Means` 算法进行无监督聚类。
   - 自动提取类簇内的核心词作为标签，以树状图 (Tree Graph) 的形式展示文本的宏观语义结构。

4. **中英文档兼容**
   - 底层集成 `jieba` 分词，兼容纯中文、纯英文以及中英混排文档的文本解析。

---

## 环境与依赖

* **后端框架**: `Flask`
* **NLP 与算法**: `jieba`, `scikit-learn`
* **图像生成**: `wordcloud`, `matplotlib`, `Pillow`
* **前端资源**: `ECharts 5`, `Tailwind CSS`

**安装依赖**：
请在项目根目录下运行以下命令安装所需环境：
```bash
pip install -r requirements.txt
```

---

## 使用说明

1. **启动服务**
   在终端运行主程序：
   ```bash
   python app.py
   ```
   控制台将输出服务运行地址（通常为 `http://127.0.0.1:5000`）。

2. **操作流程**
   - 在浏览器中打开对应的本地地址。
   - 点击 **“选择 TXT 文件”** 上传待分析的文本。
   - （可选）输入特定需要监测的关键词，或设置停用词（以逗号分隔）。
   - 在下拉菜单中选择界面配色主题。
   - 点击 **“开始计算”**，等待数据处理完毕后即可查看图表。
